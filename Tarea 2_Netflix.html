Análisis de Netflix: exploración y limpieza¶

Este notebook aplica el mismo flujo que usamos en Intro_Titanic.ipynb, pero con el catálogo de Netflix (netflix_titles.csv).

Pasos:

  1. Cargar y explorar los datos.
  2. Detectar nulos, duplicados y anomalías.
  3. Limpiar (rellenar nulos, corregir anomalías).
  4. Crear columnas nuevas (feature engineering).
  5. Graficar hallazgos.

7. Rellenar valores faltantes¶

  • country → moda (equivalente a Embarked en Titanic).
  • rating → moda (incluye los que corregimos arriba).
  • director, cast → no tiene sentido usar una moda (son nombres propios muy variados), así que usamos "Desconocido".
  • date_added → "Desconocido".

8. Crear columnas nuevas (feature engineering)¶

Equivalente a family_size, is_alone, title, has_cabin, age_group del Titanic:

  • is_movie: 1 si es película, 0 si es serie.
  • duration_minutes: minutos, solo para películas (extraído del texto de duration).
  • n_seasons: número de temporadas, solo para series.
  • main_country: primer país listado en country (algunos títulos tienen varios separados por coma).
  • n_genres: cuántos géneros aparecen en listed_in.
  • decade: década de release_year.
  • has_director: 1 si se conoce el director, 0 si es "Desconocido".
In [26]:
df["is_movie"] = (df["type"] == "Movie").astype(int)
In [27]:
def extraer_minutos(row):
    if row["type"] == "Movie" and pd.notna(row["duration"]):
        return int(row["duration"].split(" ")[0])
    return np.nan

def extraer_temporadas(row):
    if row["type"] == "TV Show" and pd.notna(row["duration"]):
        return int(row["duration"].split(" ")[0])
    return np.nan

df["duration_minutes"] = df.apply(extraer_minutos, axis=1)
df["n_seasons"] = df.apply(extraer_temporadas, axis=1)
In [28]:
df["main_country"] = df["country"].str.split(",").str[0].str.strip()
In [29]:
df["n_genres"] = df["listed_in"].str.split(",").apply(len)
In [30]:
df["decade"] = (df["release_year"] // 10 * 10)
In [31]:
df["has_director"] = (df["director"] != "Desconocido").astype(int)
